# Vagrant-first
first vagrantfile 입니다



## disk size
disk size 설정하기 위해서는 다음을 설치해야합니다.

    vagrant plugin install vagrant-disksize


**vm 이 올라간뒤 약간의 시간이 지나야 ssh 접속 이 가능합니다.**

## vagrant command

```
vagrant up 
vagarnt status
vagrant ssh [vm name]
vagarnt destroy
vagrant reload  # vm 재실행, vagrantfile에 수정된 내용 반영
```
